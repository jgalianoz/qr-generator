## About The Project

An extension that you can use to generate a QR Code.

![Demo](assets/images/features.gif)

## Getting Started

Install the dependencies:

```sh
$ npm install
// or
$ yarn
```

## Built With

- JavaScript
- [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator)
